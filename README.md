# Dither Paint

Dither Paint is an online app for drawing grayscale dither patterns.

## Online app

The app is available [online](https://mcdevon.github.io/dither-paint/).

## Features

- Draw dither patterns with easy paint tools
- Undo with ctrl+z
- PNG export
